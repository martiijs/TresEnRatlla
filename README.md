## Breu descripció del funcionament
Aquest projecte implementa el joc de "Tres En Ratlla" (també conegut com a "Tic-Tac-Toe") utilitzant C++. 
- El joc es juga en un tauler de 3x3 on dos jugadors alternen torns per col·locar les seves fitxes (X o O).
- Guanya el primer jugador que aconsegueixi alinear tres fitxes en línia recta (horitzontal, vertical o diagonal).
- El programa permet jugar una partida completa i detecta situacions de victòria o empat.
